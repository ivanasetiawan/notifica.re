The scenario above is no magic nor a science fiction story, it is about how the future is foreseen by enthusiastic nerds behind Notificare team. They truly believe that the world can be a better place when automation and machine intellegence help humans in doing their daily activities so that we can shift from spending hours to do chores to doing pivotal or fun stuff we enjoy.

With this in mind, Notificare is shaped and refined. Here is how this awesomeness started.

~ Not if I care! ~

"Notificare was born out of neccesity, the idea came up since the web 2.0", Joel started. Notificare team believes that the web and the future of it will not be about people inserting, collecting, or manupulating data but will be about automate how the data is fetched.

We are entering an age where everything is written in a way that is open to the outside world so any system can access. Notificare idea started growing from having the necessity of connecting to an API in some system and at the same time connecting to another API on other system and being able to manipulate that data by doing some easy action such as clicking a button or by just automating, identifying that the data is what we need
